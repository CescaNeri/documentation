    Common Types

    EntityId

    EntityId represents how the identity of entities is passed in the system enabling the possibility to distinguish between one another of the same kind.

    Percentage

    percentage represents the level of a source as number in [0,1]. Constraints:

    • \(0 <= base1Value <= 1\)
    • \(base100Value = round(base1Value * 100)\)

    Speed

    metersPerSecond and kilometersPerHour are two different units for the same speed value. Constraints:

    • \(metersPerSecond = kilometersPerHour / 3.6\)
    • \(metersPerSecond >= 0\)
